# Match3 Lite

A simplified browser-based match-3 game inspired by Candy Crush core mechanics:
- Swap adjacent tiles
- Match 3+ to clear
- Cascades grant bonus points
- Fixed moves per run
- Player-name leaderboard

## Research notes (inspiration)
Most popular match-3 references emphasize:
1. Simple swap-to-match core loop
2. Cascading chain reactions
3. Special rewards from larger matches (simplified here as score bonuses)
4. Short session goals (moves limit)

Sources used:
- https://en.wikipedia.org/wiki/Candy_Crush_Saga
- https://www.gameanalytics.com/blog/how-to-crack-the-match-3-code-part-1

## Run
Open `index.html` in a browser.
